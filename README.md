# 开机启动管理器

> 版本：**v1.0.1** ｜ 项目地址：https://github.com/hebulin/win_startup_manager

一个 Windows 图形界面小工具，用于把任意 `.exe` / `.bat` / `.cmd` 程序添加为**开机自启**，并随时查看 / 删除已添加的启动项。

所有操作都在 `HKEY_CURRENT_USER`（当前用户）下完成，**不需要管理员权限、不会弹 UAC**。

---

## ✨ 功能特性

- 🖱️ **纯图形界面**：选目录 → 扫描 → 添加，全程鼠标操作，无需命令行。
- 🔍 **递归扫描**：扫描时自动遍历所有子文件夹，并以**进度条**展示扫描进度。
- 🔓 **免管理员权限**：仅写入 `HKCU` 注册表，普通用户即可使用。
- 🏷️ **别名支持**：可给启动项起一个好记的名字，记录保存在 `aliases.json`。
- ✅ **二次确认**：添加 / 删除启动项前都会弹出确认框，展示名称与路径，防止误操作。
- 📋 **一键查看/取消**：以表格形式列出所有开机启动项，勾选即可删除。
- 🗂️ **删除历史**：删除的启动项会记录到 `deleted_startup.json`，便于日后追溯。
- 🛡️ **健壮不闪退**：注册表键不存在、文件不存在等异常已统一捕获。

---

## 🚀 快速开始

直接双击 **`startup_manager.exe`** 运行即可。

> 该 exe 使用 `--windowed` 打包，运行时**不会弹出黑色控制台窗口**。

## 🖼️ 界面说明（四个选项卡）

程序主界面采用选项卡布局，分为「设置开机启动应用」「取消开机启动应用」「应用取消历史」「关于」四个页签。

### 1️⃣ 设置开机启动应用
- **目录选择**：点击「浏览...」选目录 → 点击「扫描」。
- **候选文件**：扫描时会**递归遍历所有子文件夹**，并以**进度条**展示进度；结果区显示找到的 `.exe` / `.bat` / `.cmd`（完整路径）。**点选一项**，会自动把文件名（去扩展名）填入下方的「别名」框。
- **设置别名**：输入/修改别名 → 点击「添加为开机启动」。别名非空则用别名作启动项名称，否则用文件名（去扩展名）。点击后会**弹窗二次确认**（显示名称与路径），确认后才写入，并自动刷新「取消」页列表。

### 2️⃣ 取消开机启动应用
- 表格四列：**启动项名称 / 程序路径 / 别名 / 备注**。
- 「刷新列表」重新读取注册表 `Run` 键与 `aliases.json`。
- 选中一行点「取消选中启动项」会**弹窗二次确认**，确认后才删除；删除的项会自动记录到「应用取消历史」。

### 3️⃣ 应用取消历史
- 表格四列：**启动项名称 / 程序路径 / 别名 / 删除时间**，按删除时间倒序展示。
- 「刷新历史」重新读取 `deleted_startup.json`；「清空历史」二次确认后清空全部历史。

### 4️⃣ 关于
- 显示版本号 **v1.0.1** 与项目地址。
- 点击蓝色项目地址链接，或点「在浏览器中打开」按钮，会**调用系统默认浏览器**打开项目主页；「复制地址」按钮可把地址复制到剪贴板。

## 📁 数据存储位置

| 数据 | 位置 |
|------|------|
| 启动项（注册表） | `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run` |
| 别名记录（文件） | 与 **exe 同目录** 的 `aliases.json` |
| 删除历史（文件） | 与 **exe 同目录** 的 `deleted_startup.json` |

> 💡 **重要提示（关于 `aliases.json` 的位置）**：
> exe 采用 **onefile 单文件** 模式打包。程序已做兼容处理：**运行时会以 `startup_manager.exe` 自身所在目录作为基准**，因此 `aliases.json` 会稳定生成在 **exe 旁边**。
> ✅ 请把 `startup_manager.exe` 放到一个你有写入权限、且固定不变的目录（如 `D:\Tools\` ），不要长期放在临时下载文件夹里，避免别名记录随 exe 一起被清理。
> ⚠️ 切勿把 exe 放在系统保护目录（如 `C:\Program Files`），否则当前用户可能没有写入权限，导致 `aliases.json` 保存失败。

## 📝 别名说明

- `aliases.json` 结构为 `{ "启动项名称": "别名" }`。
- 添加启动项时：**别名非空 → 用别名；别名留空 → 用文件名（去扩展名）** 作为注册表里的启动项名称。
- 删除某启动项时，对应的别名记录会一并从 `aliases.json` 中移除。

---

## ⚠️ 使用注意

1. **路径是绝对路径**：添加时记录的是被选程序的当前完整路径。如果你**移动或重命名**了被添加的程序，开机启动会失效，需要重新添加。
2. **名称不能重复**：启动项名称在注册表里是唯一键，若新名称与已有项重名，会覆盖旧项。
3. **删除其他程序的启动项请谨慎**：本工具会列出 `Run` 键下的**所有**开机启动项（包括非本工具添加的），删除前请确认。
4. **杀毒软件拦截**：个别杀毒软件可能对"修改开机启动项"行为敏感，若 exe 被误报，请加入信任列表即可。

## 🔧 从源码运行 / 重新打包

需要本机已安装 Python 3（建议 3.8 及以上）。

```bash
# 1) 直接运行源码
python startup_manager.py

# 2) 安装打包工具
pip install pyinstaller

# 3) 打包成单文件 exe（无控制台窗口）
pyinstaller --onefile --windowed startup_manager.py
```

打包产物在 `dist\startup_manager.exe`，可直接拷走分发。

## ❓ 常见问题

**Q：双击 exe 没反应 / 闪退？**
A：多半是杀毒软件拦截，或 exe 放在无写入权限的目录。请放到普通用户目录（如桌面、`D:\Tools\`）后再试。

**Q：换台电脑别名没了？**
A：别名存在当前电脑的 `aliases.json`（exe 旁边）和注册表中，换电脑不会自动同步，需在新电脑上重新添加。

**Q：怎么彻底卸载？**
A：先在程序里删除所有启动项，再删除 `startup_manager.exe` 和同目录的 `aliases.json` 即可。
